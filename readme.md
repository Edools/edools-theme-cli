# Usage
<!-- usage -->
```sh-session
$ npm install -g edools-theme-cli
$ edools-theme-cli COMMAND
running command...
$ edools-theme-cli (-v|--version|version)
edools-theme-cli/3.1.0 darwin-x64 node-v10.15.1
$ edools-theme-cli --help [COMMAND]
USAGE
  $ edools-theme-cli COMMAND
...
```
